두산 베어스 온라인샵 크롤링

2019-04-02
crawling, python


크롤링(Crawling)이란
웹페이지를 그대로 가져온 뒤,
가져온 내용을 기반으로 데이터를 추출해 내는 행위를 뜻한다
개인이 대량의 웹사이트 정보를 가져올 때, 수작업을 하기에 작업량이 많을 때,
크롤러를 이용한다


[크롤링 매커니즘]은
대략적으로 이하와 같다

1) 대상 선정
URL 선정 과정
2) 데이터 로드
API라면 XML,JSON 문서 다운,
웹 페이지라면 HTML 문서를 다운받는 것
3) 데이터 분석
어떠한 부분을 수집할지 선정
4) 데이터 수집
선정한 부분을 추출해 파일/데이터를 메모리상에 저장하는 과정



실습을 위해
두산 베어스 온라인샵 이미지를 크롤링 해보자!



대상 선정


먼저 크롤링을 원하는 페이지에 접속한다
(나는 콜라보 굿즈 악개이므로 콜라보레이션 페이지를 크롤링하겠다!)


데이터 로드


우선 python에서 웹상의 데이터를 처리할 때 필요한 모듈 requests와
HTML,CSS,JavaScript 등의 웹 언어의 가독성을 높이기 위한
BeautifulSoup을 import한다


첫 페이지부터 크롤링을 진행하도록,
page 변수에 1을 저장하고,
페이지가 최대 크롤 페이지보다 적을 동안 크롤링을 진행하도록,
while문을 사용한다
(최대 페이지는 다음과 같이 7이다)

크롤할 사이트의 주소와 페이지 번호를 합쳐 저장한 값을
변수 url에 저장한다
타겟 url의 데이터는 requests.get(url)을 통해 불러오며,
이렇게 받은 데이터를 임의 변수 source_code에 저장한다


source_code의 텍스트 부분만 plain_text에 저장하고,
BeautifulSoup을 사용해 가독성을 높인 데이터를 변수 soup에 저장한다
(lxml이라는 parser를 사용)


실행 결과,
가독성이 좋은 html 코드가 나온다



데이터 분석



상품 리스트와 이미지를 크롤링해보자
각 상품 부분의 html 코드를 확인한다
(크롤링할 홈페이지에서 F12를 누르면 확인 가능하다)

상품 리스트를 크롤링할 땐,
class가 tit_goodsnm인 span태그 이용하고
이미지를 크롤링할 땐, img 태그를 이용해보자 :)



데이터 수집


1. 상품 리스트 크롤링
findAll을 통해
class가 tit_goodsnm인 span을 모두 찾아 변수 nameList에 저장한다
nameList의 텍스트만 추출해 출력한다


크롤링된 리스트들이 완벽하게 완료되었는지,
결과의 끝 부분과 크롤링 타겟의 마지막 페이지를 비교한다

크롤링이 제대로 완료되었음을 알 수 있다!


2. 상품 이미지 크롤링
타켓이미지를 크롤링해서 저장하는 코드를 짜보자



코드를 살펴보면,
src = “../data/goods/filename.jpg”의 형식으로 되어있다
..로 생략된 주소를 변수 URL에 저장한다
타겟 이미지를 저장할 path는 outpath에 저장한다
타켓 url의 html 코드에서 img 태그를 모두 찾고,
이미지의 src를 추출한다
이때, src의 ../부분은 슬라이싱을 통해 잘라준다


requests.get(image_url).content를 통해
서버 응답을 받아 파일 내용을 획득한다
content는 응답받은 RawData다
os.path.basename(image_url)을 사용해
URL에서 파일명을 획득한다
urllib.request.urlretrieve(url,path+filename)을 사용해 저장한다
이는 url이 가리키는 주소에 접근해,
해당 자원을 로컬 컴퓨터에 copy하는 기능을 한다

코드 실행 결과,
지정 path에 다음과 같이 이미지가 저장된다
