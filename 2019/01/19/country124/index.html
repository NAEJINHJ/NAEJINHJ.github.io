파라미터 n을 3으로 나눈 나머지 (n%3)
n % 3 == 0 인 경우는 예외 처리해야 함
(n=15일 경우)
> 이 경우 n-1 값을 n으로!

나머지 값을 인덱스로 리스트에 접근하기 위해,
리스트의 값은 [4,1,2]로 설정했음
